# binganime-Api
### install
1. git clone https://github.com/rishabhkr-r111/api-binganime.git
2. ```cd api-binganime```
3. ```python -m .venv .```
4. ```.venv\Scripts\activate.bat```
5. ```pip install -r requirements.txt```
6. ```flask --app api run```
